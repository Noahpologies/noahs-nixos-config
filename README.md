# My NixOS Config, started 6/21/2026.

Flake-based NixOS with home manager. Not sure what else to say really. Recently I added a tiny file that lets you choose a KDE theme, I only have one at the moment but I'm working on making some more. Run the command switcheroo to open this file and uncomment the desired config before rebuilding. There's probably a cleaner / better way to do this but I'm too lazy to look for one.
